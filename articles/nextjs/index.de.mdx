# Next.js

Next.js ist ein von **Vercel** entwickeltes Framework auf Basis von React, das es dir ermöglicht, blitzschnell performante Web-Apps zu erstellen.

- **Server-Side Rendering (SSR):** HTML wird pro Anfrage auf dem Server gerendert – ideal für dynamische Seiten und SEO.  
- **Static Site Generation (SSG):** Inhalte werden zur Build-Zeit als statische Dateien ausgegeben – ultraschnell per CDN auslieferbar.  
- **Incremental Static Regeneration (ISR):** Statische Seiten können nachträglich (on demand oder zeitgesteuert) aktualisiert werden, ohne den gesamten Build neu zu fahren.  
- **App & Pages Router:** Wähle zwischen dem etablierten Pages Router oder dem modernen App Router mit React Server Components.  
- **Integriertes Routing:** Dateibasiert – jede Datei unter `app` oder `pages` wird automatisch zur Route.  
- **API Routes & Edge Functions:** Baue serverlose Funktionen direkt im selben Repo.  
- **Automatisches Code-Splitting & Image-Optimization:** Bessere Ladezeiten ohne händische Konfiguration.  

**Ergebnis**: Kürzere Time-to-Interactive, verbesserte SEO und weniger Boilerplate – alles out-of-the-box.
