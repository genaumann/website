# Tolgee

Tolgee ist eine **vollständige Lokalisierungsplattform**, die den gesamten Lebenszyklus deiner Übersetzungen abdeckt – vom String-Management bis zum Live-Editing im Browser.  
Dank SDK, CLI und REST-API lässt sich Tolgee in praktisch jede Technologie integrieren.

- **In-Context-Übersetzung**: Texte direkt in der laufenden Anwendung bearbeiten.  
- **Maschinelle Vorschläge**: DeepL, Google, Azure … per Klick akzeptieren.  
- **Translation Memory**: Wiederverwendet vorhandene Formulierungen automatisch.  
- **Automatische Screenshots**: Jede Zeichenkette bekommt einen UI-Schnappschuss.  
- **Self-Hosted oder Cloud**: Docker-Compose in < 1 Min oder fertig gehostet.  
- **SDKs**: React, Angular, Vue 3, JS, iOS, Android, Flutter, …  

Jede Zeichenkette wird über einen **Key** referenziert – die Übersetzungen dazu pflegst du in Tolgee.  
Für die Auslieferung speichert Tolgee alle Strings als kompakte JSON-Dateien, die deine App zur Laufzeit lädt.

## Funktionsweise in Next.js

Mit `@tolgee/react` kanns du Tolgee in Next.js integrieren.

Tolgee unterstützt sowohl **App Router** als auch **Pages Router** und funktioniert client- wie serverseitig (RSC).
