# Next.js

## de

Meine ersten React-Projekte habe ich mit Create React App (CRA) umgesetzt.
Als das Framework nicht mehr zeitgemäß war, bin ich auf Next.js gestoßen – und möchte seitdem nicht mehr darauf verzichten.

Bei Webprojekten setze ich konsequent auf Next.js, das Hosting erfolgt über die Plattform Vercel.
Auch diese Webseite basiert auf Next.js und wird über Vercel bereitgestellt.

## en

I started my first React projects using Create React App (CRA).
When the framework became outdated, I switched to Next.js—and haven’t looked back since.

Today, I rely exclusively on Next.js for all web projects, with hosting handled via the Vercel platform.
This very website is also built with Next.js and hosted on Vercel.
