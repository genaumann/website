# Kubernetes

## de

Ich habe 2021 erstmals mit Kubernetes gearbeitet und dabei gelernt, wie Anwendungen in bestehende Cluster integriert und dort lauffähig gemacht werden.
In der Umgebung wurde Argo CD für GitOps eingesetzt, wodurch Deployments automatisiert und versioniert abliefen.
Diese ersten praktischen Erfahrungen haben mein Verständnis für containerisierte Anwendungen und moderne Deployment-Strategien stark geprägt.

## en

I was first introduced to Kubernetes in 2021, where I learned how to deploy and run applications within existing clusters.
The environment used Argo CD for GitOps, enabling automated and version-controlled deployments.
These early hands-on experiences helped me build a solid understanding of containerized applications and modern deployment strategies.
