# TypeScript

## de

TypeScript nutze ich inzwischen fast täglich, um Webseiten zu programmieren (Front- und Backend) und um das End-to-End-Testframework Playwright zu bedienen.

TypeScript kann sowohl im Frontend als auch im Backend eingesetzt werden und bietet robuste Typisierung mit Unterstützung durch moderne IDEs.

## en

I use TypeScript almost daily to develop websites (Frontend and Backend) and to operate the End-to-End testing framework Playwright.

TypeScript can be used in both Frontend and Backend and offers robust typing support with modern IDEs.
