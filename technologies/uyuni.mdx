# Uyuni

## de

2023 hatte ich erste Berührungspunkte mit Uyuni / SUSE Manager – einer Software, die Salt zur Serververwaltung nutzt.

Im Rahmen eines Kundenprojekts habe ich den SUSE Manager und Salt so erweitert, dass sowohl ein Infrastructure-as-Code-(IaC)-Ansatz als auch ein Staging-Konzept umgesetzt werden konnten.

## en

In 2023, I had my first hands-on experience with Uyuni / SUSE Manager—a tool that uses Salt for server management.

As part of a client project, I extended SUSE Manager and Salt to support an Infrastructure as Code (IaC) approach and implement a staging concept.
